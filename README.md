# WebMoji

WebMoji is a real-time hand gesture recognition desktop application developed with C# and WPF.

The application uses webcam input to detect and count fingers, then displays corresponding images based on the detected hand gesture.

## Features

- Real-time webcam processing
- Finger counting system
- Hand gesture recognition
- Dynamic image display
- Desktop UI built with WPF
- Modular project architecture

## Technologies Used

- C#
- WPF
- Computer Vision
- .NET

## Project Structure

- WebMoji.Core — core application logic
- WebMoji.Vision — hand detection and image processing
- WebMoji.Desktop — WPF desktop interface

## How It Works

1. The webcam captures live video.
2. The application processes frames in real time.
3. Fingers are detected and counted.
4. An image matching the detected number is displayed on screen.
